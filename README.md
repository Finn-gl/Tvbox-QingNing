## 3. 想做自己的接口？
>这里只针对没有自己服务器的宝子们，下面是一些常见的开源仓库
```
https://gitee.com/
https://coding.net/
https://gitcode.net/
https://gitlab.com/
https://gitea.com/
https://notabug.org/
https://github.com/
```
>**特别注意**：仓库名不要以"jiekou"、"tvbox"、"box"等敏感字眼命名，这样有可能会被"列如黑名单"、"封禁账号"等，最好把仓库体积弄大一点，加点杂七杂八的东西进去，这样打造一个属于你的自用接口是没问题的
